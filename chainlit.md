# 🔬 Project 4: AI Agent Security Research Assistant

Welcome to the **Multi-Agent Cybersecurity Research System** — powered by **OpenClaw** + **Google Gemini**.

## 🏗️ Architecture
This system uses **7 specialized AI agents** working together:

| Agent | Role |
|-------|------|
| 🎯 **Coordinator** | Plans and orchestrates research tasks |
| 🔍 **Search-A** | Broad web & academic research |
| 🔎 **Search-B** | Technical CVE & exploit research |
| ✅ **Verification** | Cross-references and fact-checks findings |
| 📊 **Ranking** | Ranks findings by relevance & credibility |
| 📝 **Synthesis** | Produces the final research report |
| 💾 **Data** | File I/O operations |

## 🚀 How to Use
Type a cybersecurity research question and watch the agents collaborate in real-time!

**Example queries:**
- *"Analyze prompt injection defenses in LLM-based agents"*
- *"Research supply chain attacks on AI model registries"*
- *"Compare detection methods for adversarial machine learning attacks"*
